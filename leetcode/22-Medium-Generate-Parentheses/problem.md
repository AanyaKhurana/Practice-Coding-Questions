# Problem 22: Generate Parentheses


#### Difficulty: Medium

#### Problem

Given n pairs of parentheses, write a function to generate all combinations of
well-formed parentheses.

#### Example

<pre>

For example, given n = 3, a solution set is:

```
[
  "((()))",
  "(()())",
  "(())()",
  "()(())",
  "()()()"
]
```

</pre>
