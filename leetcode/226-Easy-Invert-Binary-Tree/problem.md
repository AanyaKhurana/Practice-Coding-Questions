# Problem 226: Invert Binary Tree


#### Difficulty: Easy

#### Problem

Invert a binary tree.

#### Example

<pre>
     4
   /   \
  2     7
 / \   / \
1   3 6   9

Output:

     4
   /   \
  7     2
 / \   / \
9   6 3   1
</pre>
